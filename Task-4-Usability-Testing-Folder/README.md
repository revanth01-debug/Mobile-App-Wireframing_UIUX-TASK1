## Project Overview

This task focuses on conducting usability testing for the QuickBite Food Delivery Application prototype. The objective was to evaluate the user experience, identify usability issues, gather user feedback, and recommend design improvements.

---

## Objective

- Evaluate the usability of the QuickBite mobile application.
- Observe user interactions and navigation patterns.
- Identify pain points and usability issues.
- Collect feedback from sample users.
- Recommend improvements based on findings.

---

## Product Tested

QuickBite Food Delivery Mobile Application Prototype

---

## Participants

| User Type | Age |
|------------|-----|
| Student | 20 |
| Working Professional | 28 |
| Food Enthusiast | 24 |

---

## Testing Method

- Task-Based Testing
- User Observation
- Feedback Collection
- Usability Analysis

---

## Test Scenarios

1. Login to the application
2. Browse food categories
3. View food details
4. Add an item to cart
5. Access profile section

---

## User Feedback

### Student (Age 20)

**Positive Feedback**
- Easy navigation
- Attractive interface
- Quick access to categories

**Suggestion**
- Search bar could be larger

---

### Working Professional (Age 28)

**Positive Feedback**
- Fast ordering process
- Clean design
- Easy food discovery

**Suggestion**
- Cart icon needs better visibility

---

### Food Enthusiast (Age 24)

**Positive Feedback**
- Appealing food images
- Smooth browsing
- Modern interface

**Suggestion**
- Add favorites/wishlist feature

---

## Findings & Analysis

### Positive Findings

- Users completed tasks successfully
- Navigation was easy to understand
- Modern and attractive interface
- Buttons were clearly visible

### Issues Identified

- Search feature needs better visibility
- Cart icon could be more prominent
- Wishlist functionality is missing

### Success Rate

**85% Overall Usability Success Rate**

---

## Design Improvements

- Increase search bar visibility
- Add wishlist/favorites functionality
- Improve cart accessibility
- Enhance category discoverability

---

## Recommendations

- Continue usability testing in future versions
- Improve navigation efficiency
- Enhance search experience
- Introduce personalization features

---

## Tools Used

- Figma

## Outcome

The usability testing demonstrated that users were able to navigate the QuickBite application successfully and complete important tasks with minimal assistance. Feedback collected during testing helped identify areas for improvement and provided valuable insights for future iterations.
